User Guide

1. Installation

Once installed, the extension works silently in the background. No configuration is required to start basic protection.

2. Automatic Detection

Navigate to any website with a password field (e.g., Gmail, GitHub). When you click the password box, our Security UI will appear automatically.

3. Reading the Metrics

  • Entropy: Measures randomness. Higher is better (aim for 60+).
  • Crack Time: Estimated time to break the password using brute force.
  • Breach Status: If red, change your password immediately.

Security Architecture

Zero-Knowledge Architecture

We never store, transmit, or log your passwords. All analysis happens locally in your browser memory.

K-Anonymity Model (HIBP)

When checking for breaches, we never send your full password. We use the SHA-1 hash and send only the first 5 characters to the API.

Request: https://api.pwnedpasswords.com/range/21BD1
Local Match: Checking remaining characters against response list.

This ensures the server can never identify your specific password.
